# Restaurant Menu

Online restaurant menu built with HTML, CSS, and JavaScript.

## Features

- Mobile-friendly design
- Fast loading
- QR code accessible
- Easy to update menu items and prices

## Project Structure

```text
├── index.html
├── style.css
├── script.js
└── images/
```

## Live Website

Replace with your GitHub Pages URL:

https://YOUR_USERNAME.github.io/restaurant-menu/

## QR Code

Generate a QR code that points to the live website URL above and place it on tables for customers to scan.
